##  Como jogar  

- ⬅️ ➡️ → mover jogador  
- ⬆️ / Espaço → pular
- 🅩 → atirar  
- 🅜 → ligar/desligar som  
- ⬆️ ⬇️ + Enter → navegar no menu  
- 🅡 → reiniciar  
- 🅧 ou Esc → sair  



##  Objetivo  

- Derrote **10 inimigos** para vencer.  
- Se cair da tela ou perder todas as vidas → **Game Over**.

✔️  sons ,<br>
✔️ score de 10 inimigos mortos = victory,<br>

## 🔊 Sons

🎵 `jump.wav` → pulo  
🎵 `laser.wav` → tiro
